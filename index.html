HandiCricket

How to Play:

  1. One player creates a game and shares the room code
  2. The other player joins using the room code
  3. Complete toss and team selection
  4. Play by selecting numbers (0-6) each ball
  5. If numbers match, it's a wicket!
  6. If batter plays 0, bowler's number is scored
  7. If bowler plays 0, it's a dot ball
  8. If both play 0, it's a no ball + free hit
