How to Play:

  1. Create or Join a match
  2. Complete the toss to decide batting order
  3. Select your playing 11 squad
  4. Play by selecting numbers (0-6) each ball
  5. Matching numbers = Wicket!
  6. Batter's 0 = Bowler's number is scored
  7. Bowler's 0 = Dot ball
  8. Both 0 = No ball + Free hit
